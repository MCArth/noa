        getTerrainMatId

        • getTerrainMatId(blockMatID: any): any

        • Maps a given matID (from noa.registry) to a unique ID of which terrain material can be used for that block material. This lets the terrain mesher map which blocks can be merged into the same meshes. Internally, this accessor also creates the material for each terrainMatID as they are first encountered.
